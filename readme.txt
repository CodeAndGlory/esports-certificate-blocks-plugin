== Description ==

This plugin adds independent block functionality specifically for the esports-certificate-yootheme-child theme which handles presentation.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the plugin files to the `/wp-content/plugins/esports-certificate-blocks-plugin` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
